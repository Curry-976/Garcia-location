# Garcia Location

Landing page pour Garcia Location — service de location de véhicules.

## Structure

- `Garcia Location.html` — point d'entrée HTML
- `app.jsx` — composant racine
- `landing.jsx` / `landing2.jsx` — sections landing page
- `heromock.jsx` — section hero
- `prebooking.jsx` — formulaire de pré-réservation
- `widget.jsx` — widget de réservation
- `ios-frame.jsx` — cadre de maquette iOS
- `tweaks-panel.jsx` — panneau de personnalisation
- `styles.css` / `sections.css` / `sections2.css` / `heromock.css` — styles
- `data.jsx` — données statiques
- `icons.jsx` — icônes SVG
